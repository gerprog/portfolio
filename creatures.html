Crazy Creatures

Crazy Creatures is a lively and fun game for young children. The goal is to create creatures by answering questions and doing tasks.

This is a college project with a team of 3.

The jigsaw with draggable elements encourages constructive learning. The animals could be replaced by geographic or interstellar maps, body parts etc.

The final stage is a bonus where the player can randomly select and create new creatures in full or simply have fun with some unexpected results.

I created the creatures using only Adobe Photoshop.

Technology used:

php5, CSS3, Adobe CS.
JQueryUI for the drag and drop functionality.
JavaSCript and JSON for the quiz elements.
