Snappo!

A college project built with Ruby on Rails, Snappo! suggests photo challenges to the user.

Comprising of a few tasks, these challenges can be accepted and interpreted in any way. The results are uploaded for view in the gallery and to be voted on by other users.

This solo project evolved through 5 iterations each of which involved user testing.

Technology used:

HTML5, CSS3, Adobe CS
Ruby on Rails
SQLite3
Bootstrap
JQuery
AJAX
Ruby gems Paperclip, ThumbsUp, WillPaginate.
Disqus for the forum.
